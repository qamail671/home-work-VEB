Git

  • git init - создание нового реппозитория GIT в пустой дирректории или превращение существующей дирректории в новый реппозиторий GIT.
  • git push - выгрузка сделанных изменений в удаленный реппозиторий.
  • git pull - получение изменений из удаленного реппозитория на локальную машину.
  • git status - просмотр списка измененных файлов.
  • git commit -m - сохранение изменений в git.
